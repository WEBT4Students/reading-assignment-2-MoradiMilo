Identify void elements and how they are denoted

A void element is an element whose content model never allows it to have contents under any circumstances. Void elements can have attributes. . Example of such elements are br, hr, img, input, link, base, meta, param, area, embed, col, track, source etc.

Identify attributes

All HTML elements can have attributes Attributes provide additional information about elements Attributes are always specified in the start tag Attributes usually come in name/value pairs like: name="value"
